# EN

### How to use open source projects?
• Download the open-source code and upload it to your own repository. Open the file, click "raw" to get the raw page link, then use loadstring(game:HttpGet("Link"))() to launch the script.


### Open Source Project Important Notes!
• We only handle open-source/cracking work and do not provide any issue fixes (including but not limited to bugs, feature failures, scripts failing to launch, etc.). Projects will not be maintained long-term. Please resolve any issues on your own! Do not send us related emails/messages (or spam) regarding these matters!
