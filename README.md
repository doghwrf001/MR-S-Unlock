# EN
### How to use cracked projects?

• Open the file, click "raw" to get the raw page link, then use loadstring(game:HttpGet("Link"))() to launch the script.

• Still don't know how to use it? No worries! Open the script file you want to play—it already contains the ready-to-use code. Just copy and paste it into your executor to use!
